# User API Spec

## Register User

Endpoint : POST /api/users

Request Body :

```json
{
  "username": "ewin",
  "password": "pass123",
  "name": "ewin pirade"
}
```

Response Body :

```json
{
  "username": "ewin",
  "password": "pass123",
  "name": "ewin pirade"
}
```
